# Overview

The purpose of this app is to provide an interface for visualizing bits and operations on or between them. The need for this app comes from me being dissatisfied with functionality of the currently available bit calculators and visualizers.

## Functionality

The app should provide the following functionality:

1. Variable Assignment
2. Position Highlighting
3. Bitwise operations
4. Bit/hex/octal Based Variable Assignment (similar to apple calculator)
5. Support for 8, 16, 32, 64, 128 sized bit sets
